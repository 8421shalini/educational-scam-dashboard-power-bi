# Online Scam Data Analysis (Power BI)

## Project Overview
This project analyzes different types of online scams using Power BI.

## Dataset
The dataset contains:
- Scam types
- Loss amount
- Victim age group
- Country
- Year of incidents

## Dashboard Features
- KPI cards for total scams and loss
- Bar chart for scam types
- Age group analysis
- Country-wise scam distribution
- Interactive slicers

## Tools Used
- Power BI
- Excel
- Data Visualization

## Insights
- Phishing scams are the most common
- Middle age groups are most affected
- Financial losses increased over time
